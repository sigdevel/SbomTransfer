#### Перед запуском:
##### 1. Создать и активировать виртуальное окружение.
```bash
python3 -m venv venv
source venv/bin/activate
```
##### 2. Установить перечисленные модули с зависимостями, указанными в файле `requirements.txt`.
```bash
pip install -r requirements.txt
```
